# Power Consilium System – UPS Solutions Website

A modern, SEO-friendly, responsive website for **Power Consilium System (PCS)** – a leading provider of UPS solutions, AMC services, and data center power infrastructure across India.

## Tech Stack

- **React 18** + **TypeScript** (Vite)
- **Tailwind CSS** – Utility-first styling with design tokens
- **Framer Motion** – Scroll-triggered animations
- **Lucide React** – Icon library

## Features

- ✅ Responsive (mobile, tablet, desktop)
- ✅ Dark mode (global `.dark` class)
- ✅ Auto/manual hero image slider
- ✅ Scroll animations (Framer Motion)
- ✅ Back-to-top button (bounce)
- ✅ Loading spinner
- ✅ SEO (meta, JSON-LD, semantic HTML, aria)
- ✅ Lazy-loaded sections
- ✅ Admin panel (`/admin`)

## Getting Started

```bash
npm install
npm run dev
```
